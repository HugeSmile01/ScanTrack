Getting Started

Welcome to the ScanTrack documentation. Here you'll find everything you need to get up and running with our attendance tracking system.

1. How to Create a QR Code

Follow these steps to generate a QR code for your students:

  1. Navigate to the Create QR Code page.
  2. Enter the required details: Student ID, Grade Level, and Student Name.
  3. Click on "Generate QR Code" to create the QR code.
  4. You can download the generated QR code by clicking on the "Download QR Code" button.

2. How to Search for Student Attendance Logs

To search for a student's attendance record, follow these steps:

  1. Navigate to the Search page.
  2. Enter the student's ID or name in the search bar.
  3. Click on "Search" to retrieve the student's attendance logs.
  4. Review the logs displayed to track the attendance history.

3. System Requirements

Ensure that you are using a modern browser that supports HTML5, CSS3, and JavaScript for the best experience. Supported browsers include Google Chrome, Mozilla Firefox, Microsoft Edge, and Safari.

4. Troubleshooting

If you encounter any issues, please refer to the following troubleshooting steps:

  • Ensure that all required fields are filled in correctly before generating a QR code.
  • If a QR code fails to generate, refresh the page and try again.
  • If the search function isn't working, double-check the student's ID or name for accuracy.
  • Clear your browser cache and cookies if you experience loading issues.
